我能做的事

以下不是全部宇宙，而是老師日常最常用、最容易立刻省時間的工作類型。每一項都附上一個實際可開口交辦的例子。

卷

出考卷與題庫

依年級、單元、難度、題型比例產生試題，並附答案與解析。

例：幫我出一份國小五年級分數加減 30 題，含 5 題應用題與答案。
課

設計教案與活動

把教學目標、時間、材料、分組方式與評量方式整理成完整流程。

例：幫我設計 40 分鐘的「能源轉換」探究課，適合八年級。
材

教材製作

產生講義、投影片大綱、學習單、閱讀文本、實驗步驟與延伸任務。

例：把這篇文章改成三種閱讀程度的學習單，附關鍵字解釋。
評

評量規準與回饋

建立 rubric、同儕互評表、作品檢核表，或把評語改得具體有幫助。

例：幫我做一份簡報評分規準，包含內容、表達、視覺與合作。
班

班級經營

協助設計班規、座位策略、獎勵制度、衝突處理話術與班會流程。

例：班上常有人打斷同學發言，幫我設計一週改善方案。
家

家長聯絡

把訊息寫得溫和、清楚、專業，並避免誤會或過度指責。

例：幫我寫一封給家長的訊息，說明孩子最近作業缺交但保留合作語氣。
差

差異化教學

同一主題可改成基礎、標準、挑戰三層任務，照顧不同程度學生。

例：把「面積」單元設計成三層任務卡，讓學生自己選擇挑戰。
文

公文、公告與行政文字

整理活動通知、會議紀錄、成果報告、申請書與校內公告。

例：幫我把校外教學注意事項改成家長容易讀懂的公告。
資

資料整理與表格分析

整理成績、問卷、出缺席、回饋意見，找出趨勢與可行下一步。

例：根據這份測驗結果，幫我找出全班最需要補救的三個概念。
站

教學網頁與互動工具

在 Antigravity 中製作簡易網頁、小測驗、抽籤器、計分板或學習任務頁。

例：幫我做一個英文單字翻卡練習網頁，可以顯示答案與重新洗牌。
語

改寫、翻譯與語氣調整

把文字改成學生版、家長版、正式版、鼓勵版，或轉成雙語教材。

例：把這段自然科說明改成國小四年級聽得懂的版本。
想

創意發想與備課討論

幫你從主題、活動、提問、情境任務到跨領域連結一起腦力激盪。

例：用「校園植物」設計一個結合自然、國語和資訊的跨域任務。

推薦老師裝的插件

Antigravity 的插件生態會隨時間改變；以下用「功能類型」推薦，安裝時可在擴充功能商店搜尋對應名稱或同類型工具。

必裝

Live Server / 本機預覽

做班級網頁、互動教材或 HTML 學習單時，能一鍵在瀏覽器即時預覽，修改後立刻看結果。

必裝

Markdown All in One

適合寫教案、會議紀錄、課程筆記與講義草稿，格式乾淨，也容易轉成其他文件。

效率

Prettier / 格式化工具

讓網頁、JSON、設定檔自動排版，學生作品或自己做的小工具都更容易閱讀與維護。

資料

Excel / CSV Viewer

快速查看測驗成績、問卷回覆、名單與分組資料，不必每次都打開大型試算表軟體。

協作

GitHub Pull Requests 或 Git 工具

若老師帶學生做專題或程式課，可管理版本、比較修改、保存作品歷程。

品質

Spell Checker / 拼字檢查

製作英文教材、雙語公告、課程網站時，能快速抓出拼字錯誤，降低發布前的檢查負擔。

怎麼跟我說話

好的提問像好的教案：對象、目標、限制、格式講清楚，產出的品質通常會立刻提升。

Do

說明學生年級、程度、課程單元與可用時間。

Do

指定輸出格式，例如表格、條列、講義、題目加答案、家長訊息。

Do

給我現有素材，並說明你想保留、刪除或改寫的部分。

Don't

只說「幫我做教材」卻沒有主題、對象或使用情境。

Don't

一次要求太多互相衝突的目標，例如又要很短、又要包含所有細節。

Don't

把學生個資、敏感家庭資訊或完整聯絡資料直接貼上來。

可以這樣問： 「我要教國小六年級『比與比例』，一節課 40 分鐘。請幫我設計一份活動式教案，包含導入、分組任務、3 題形成性評量、回家作業，語氣要適合新手老師照著做。」
